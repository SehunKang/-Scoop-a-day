## 소개
고양이의 건강관리를 위해 똥, 오줌을 체크해서 기록, 확인하는 앱입니다. 실 서비스, 학습 두가지 목적을 위해 개발하는 앱입니다.

Reactive programming을 활용하여 DB, UI의 원활한 상호작용에 중점을 두어 개발했습니다. 

고양이 이름을 test로 생성하면 약 2년치의 랜덤데이터가 생성되어 확인할 수 있습니다.
